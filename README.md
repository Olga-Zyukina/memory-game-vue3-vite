# Memory Game

A memory game created using **Vue 3 + TypeScript + Vite**.

## Features

- Select the number of cards
- Show matches
- Show turns
- Save and show the time record
- Show the game status
- Show the game time
- Adaptive design

## Technologies Used

-  Vue 3
-  TypeScript
-  Vite
-  Vue-router
-  Axios
-  Pinia

## Live Demo

Check out the live demo: [Memory Game](https://olga-zyukina.github.io/memory-game-vue3-vite/dist/)

## Project Structure

```
src/
├── components/     # Vue components
├── pages/          # Vue pages
├── types/          # TypeScript type definitions
├── services/       # Functions
├── assets/         # Static assets
└── App.vue         # Root component
```
